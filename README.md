# Bus Booking System

A web application for browsing and booking bus routes, built with Django.

## Links

- **GitHub Repository:** https://github.com/OmerJalal1/lab2-WA-Bus-site
- **Live Site:** https://web-production-695a2.up.railway.app

---

## Features

- User registration and login/logout
- Browse available bus routes
- View route details (origin, destination, duration)
- Book and cancel seat reservations
- Personal bookings page
- Admin panel for managing routes and stations

## Bonus Features

- Animated route cards with staggered fade-in delays
- PostgreSQL database for persistent storage across deployments
- Static files served with WhiteNoise

## Known Limitations

- No payment system
- No seat limit per route
- No email confirmation on booking
- Routes and stations must be added manually through the admin panel
- No search or filter functionality

## Tech Stack

- **Backend:** Django (Python)
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, Bootstrap
- **Deployment:** Railway
